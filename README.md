# JavaScript Basics Playground

This repository is a learning and experimentation project focused on JavaScript fundamentals.

The main goal of this project is to practice, test, and experiment with core JavaScript concepts in a simple and flexible environment, without the complexity of frameworks or heavy tooling.

## Purpose

This project was created to:

Refresh and strengthen JavaScript fundamentals

Experiment with language features and syntax

Test small ideas and behaviors quickly

Serve as a personal sandbox for learning

It is not intended for production use.
